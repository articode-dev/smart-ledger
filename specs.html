Product & Technical Specification Document

Product Overview

Smart Ledger is a fast, lightweight, and offline-first digital ledger application designed for both personal and business use. It replaces traditional paper khata books, allowing users to track receivables, manage customer profiles, and record transactions with absolute mathematical precision entirely on-device.

Core Features

Authentication & Setup
  • OTP-based phone number authentication
  • Support for 25+ international country codes
  • Personalized workspace setup (Name & Business)
  • Dynamic selection spanning 22 global currencies
Ledger & Transactions
  • Dual-entry tracking: Credit (Owed) vs. Payment (Paid)
  • Running balance calculated precisely from historical data
  • Quick-entry chips for rapid transaction noting
  • Distinct visual indicators (Red vs. Green) for balances

Data & Capabilities

Management Dashboard
  • Real-time aggregation of Total Receivables
  • Dynamic search filter by Name or Phone Number
  • Sorting capabilities (Balance, Name, Recency)
  • At-a-glance status chips for fast reading
Export & Portability
  • One-click automated WhatsApp message structuring
  • Text-based transaction summary generation
  • CSV Export of targeted individual ledgers
  • CSV Export of comprehensive global app data

Technical Architecture

Architecture Layer Single Page Application (SPA) / Progressive Web App (PWA)
View Engine Vanilla JavaScript (Hash-based routing system with animations)
Styling Matrix Mobile-first CSS Variables, Flexbox/Grid, Zero-dependency
Storage Paradigm Fully Offline-First (localStorage with asynchronous DB wrapper)
Network Dependency None (Service Worker intercepts and serves all cached assets)
Scalability Hooks Segmented DB and Auth modules built for seamless backend/API swapping
