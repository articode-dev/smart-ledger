Product Overview

Smart Ledger is a fast, lightweight, and fully offline-first digital ledger application for both personal and business use. It replaces traditional paper khata books, enabling users to track credit, manage customer or contact profiles, and record transactions with absolute mathematical precision — entirely on-device, with zero internet dependency.

Core Features
Authentication & Setup
  • OTP-based international phone number login
  • 25+ country codes with flag-based picker
  • Personalized workspace: Name & Business/Personal
  • 22 global currency selections with symbols
Ledger & Transactions
  • Dual-entry system: Credit (Owed) vs. Payment (Paid)
  • Auto-recalculated running balance per contact
  • Quick-note chips for fast transaction entry
  • Date picker with default set to today
  • Color-coded entries: Red = Owed, Green = Paid
Data & Capabilities
Management Dashboard
  • Global Total Receivables aggregation in real-time
  • Customer search by Name or Phone Number
  • Multi-mode sorting: Balance, Name, or Recency
  • Settled, Owed, and Overpaid status labels
Export & Sharing
  • WhatsApp share: auto-formatted text summary per contact
  • Individual contact ledger export to CSV
  • Full global data export to CSV
  • Direct call button via device dialer

Technical Architecture
Architecture Single Page Application (SPA) — Progressive Web App (PWA) installable
View Engine Vanilla JavaScript with hash-based routing and animated transitions
Storage Layer Offline-First — localStorage with async DB wrapper, fully backend-swappable
Network Dependency Zero — Service Worker intercepts and serves all cached assets offline
Scalability Hooks DB and Auth modules built for seamless backend & payment API integration
